## Installation

For development:

```bash
# from this directory
pip install -e .
```

From PyPI:

```bash
pip install pyrift
```

## Development

Use `conda` or `venv` to create and activate a virtual environment.

`pip install -e .`

## Running

Run the server with `python -m rift.server.core --port 7797`. This will listen for LSP connections on port 7797.
